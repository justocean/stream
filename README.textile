h1. stream.php

A stand-alone library for streams in PHP based off of "stream.js":http://www.streamjs.org.

Original implementation based off of the stream data structure introduced in "SICP":http://mitpress.mit.edu/sicp/full-text/book/book-Z-H-24.html section 3.5.

h2. Usage

For a more in-depth introduction to the stream datastructure, please "visit the tutorial":http://ryangantt.com/stream, which has been adapted to PHP from the well-written introduction by Dionysis.
