h1. streamphp

a tiny stand-alone library for streams in php (based off of @dionyziz "stream.js":http://www.streamjs.org)

original implementation based off of the stream data structure introduced in "SICP":http://mitpress.mit.edu/sicp/full-text/book/book-Z-H-24.html section 3.5.

h2. Usage

For usage examples, please see examples.php for now. The unit tests cover features used in examples.php.
